# MIS Performance Dashboard

**Author**  
Sahil Singh

**Overview**  
Interactive Excel dashboard to analyze sales performance across channels, regions, categories, and time. Built with Power Query, PivotTables and VBA macros for automation.

**Files in this repo**  
- `MIS_Dashboard.xlsm` — macro‑enabled workbook containing data, pivots, dashboard, and macros  
- `docs/screenshots/` — screenshots of the dashboard
- `README.md` — this file

**Workbook structure**  
- **Raw_data** — original source data (do not edit here)  
- **Cleaned_Data** — Power Query output used by pivots and charts  
- **pivot_tables** — PivotTables powering KPIs and charts
- **Dashboard** — final interactive dashboard with KPIs, charts, slicers, and buttons

**Key features**  
- Slicer‑driven KPIs   
- Interactive visuals: Monthly Sales Trend, Top Products, Sales by Region, Channel Split  
- VBA macros: `RefreshAll`, `ClearAllSlicers`
- One‑click refresh and clear filters for fast analysis
